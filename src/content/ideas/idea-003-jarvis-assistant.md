# 💡 贾维斯智能助理

## 灵感来源

钢铁侠里的贾维斯，谁不想要一个？

## 核心想法

用**香橙派5**作为大脑，运行 OpenClaw AI 代理，通过连接一个带有**麦克风和喇叭的 ESP32** 模块，实现：

- 🎤 **语音输入**：对着 ESP32 说话，实时采集音频
- 🧠 **AI 处理**：香橙派接收音频，调用 LLM 理解意图并生成回复
- 🔊 **语音输出**：TTS 合成语音，通过 ESP32 的喇叭播放
- 🏠 **指令执行**：语音控制智能家居、查询信息、执行任务

## 技术架构

```
[麦克风] → [ESP32] → WiFi → [香橙派5 (OpenClaw)]
                                    ↓
                              [LLM API]
                                    ↓
[喇叭] ← [ESP32] ← WiFi ← [香橙派5 (TTS)]
```
